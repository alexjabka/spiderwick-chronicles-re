Spiderwick Chronicles — Reverse Engineering & Modding Toolkit v1.1
==================================================================

Game: The Spiderwick Chronicles (PC, 2008) by Stormfront Studios
Engine: Proprietary "Ogre" engine, DirectX 9, Kallis VM scripting

This release contains pre-built binaries for research purposes.
Full source code available at: https://github.com/user/spiderwick-re

Contents:
  SpiderView/   — Game research tool (3D viewer, exporter)
  SpiderMod/    — ASI mod (freecam, debug overlay, scene dump)

See README.txt in each folder for setup instructions.
